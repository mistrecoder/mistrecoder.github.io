# Design Portfolio - Mehdi Rejeb Kaabia

## Style Visuel
- **Thème**: Professionnel, minimaliste, technologique
- **Approche**: Design épuré avec accent sur la lisibilité et l'impact visuel
- **Inspiration**: Design moderne tech avec touches d'ingénierie

## Palette de Couleurs
- **Primaire**: Bleu professionnel (#0044cc)
- **Secondaire**: Blanc pur (#ffffff)
- **Arrière-plan**: Gris clair (#f5f5f5)
- **Texte**: Noir doux (#1b1b1b)
- **Accent**: Bleu clair pour les interactions

## Typographie
- **Police principale**: Inter (sans-serif moderne)
- **Police secondaire**: Poppins (pour les titres)
- **Hiérarchie**: 
  - H1: 3rem (48px) - Poppins Bold
  - H2: 2.25rem (36px) - Poppins SemiBold
  - H3: 1.5rem (24px) - Poppins Medium
  - Corps: 1rem (16px) - Inter Regular
  - Petite: 0.875rem (14px) - Inter Regular

## Layout & Structure
- **Grid**: Système de grille fluide avec Tailwind CSS
- **Espacement**: Consistant avec échelle 8px
- **Container**: Max-width 1200px, centré
- **Sections**: Hero, About, Projects, Skills, Contact

## Éléments Visuels
- **Hero**: Animation de particules avec fond dégradé subtil
- **Cartes projets**: Effet hover 3D avec ombre dynamique
- **Photo profil**: Cercle parfait avec bordure bleue
- **Icônes**: Set cohérent pour les technologies
- **Boutons**: Style moderne avec transitions fluides

## Animations
- **Scroll**: Apparition progressive des sections (AOS.js)
- **Hover**: Transformations 3D sur les cartes
- **Transitions**: Timing cubic-bezier pour fluidité
- **Micro-interactions**: Boutons, liens, formulaires

## Responsive Design
- **Mobile**: Single column, touch-friendly
- **Tablette**: 2 colonnes, navigation adaptée
- **Desktop**: Layout complet avec animations avancées
- **Breakpoints**: Tailwind defaults (sm, md, lg, xl)
